我的博客演进之路(04)--调整文章版面显示大小

by · 2016年02月20日 · 121 Words · ~1min reading time | Improve on

由于原来的版面显示有点过小，在现在的宽屏面前有些浪费。于是为了更好的进行阅读显示，特做了如下修改：

打开文件./themes/rapid/static/wp-content/themes/hueman/style.css，把

  .container { padding: 0 20px; }
  .container-inner { max-width: 1380px; min-width: 1380px; width: 100%; margin: 0 auto; }

改为

  .container { padding: 0 0px; }
  .container-inner { min-width: 1380px; width: 100%; margin: 0 auto; }
