pango设计与实现(02)--快速开始

By mospan

2016年02月14日

安装

安装前准备

确保你的本地环境已经安装好如下软件：

  • git.
  • Go 1.4+

源码安装

下载源码到本地

   git clone https://github.com/yanwushuang/pango.git

设置好环境变量

   cd pango
   export GOPATH=`pwd`

编译

   make

如果编译成功，将在当前目录下生成目标可执行二进制程序：pango

安装

   make install
hugo pango
